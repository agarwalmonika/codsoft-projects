# Simple Calculator

## Overview

This is a simple calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations through an easy-to-use interface.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Decimal Calculations
- Delete Last Digit
- Clear Display
- Responsive Layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
└── README.md
```

---

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation is required.
